# ТекстПро 🛠️
**Универсальный текстовый комбайн для очистки, дедупликации и извлечения данных**

## 📖 О проекте
**ТекстПро** — это бесплатный онлайн-инструмент, объединяющий всё необходимое для быстрой и безопасной обработки текста. Не требует регистрации, установки ПО и работает полностью в вашем браузере.

## ✨ Основные функции
### 🧹 Очистка текста
Удаление технического мусора и форматирования в один клик:
- Удаление HTML-тегов (`<br>`, `<p>`, `<div>` и др.)
- Нормализация пробелов (двойные → одинарные)
- Удаление пустых строк и строк без текста
- Очистка Markdown-разметки (`*`, `#`, `>`, `~`, `` ` `` и др.)
- Удаление эмодзи и специальных Unicode-символов

### 📋 Дедупликация списков
Быстрое удаление повторов с гибкими настройками сортировки:
- Режимы: без сортировки, А → Я, Я → А, по длине строки
- Каждая строка обрабатывается как отдельный уникальный элемент

### 🎯 Лид-экстрактор
Автоматический поиск и извлечение контактных данных из произвольного текста:
- 📧 Email-адреса
- 🔗 Ссылки (URL)
- 📱 Номера телефонов (поддержка `+7`, `8`, международных форматов)

## 🔒 Приватность и безопасность
**100% клиентская обработка.** Все операции выполняются исключительно на стороне пользователя. Ваши данные **никогда не отправляются на сервер**, что гарантирует полную конфиденциальность и соответствие принципам Privacy by Design.

## 🚀 Как использовать
1. Откройте файл `ТекстПРО.html` в любом современном браузере (Chrome, Firefox, Safari, Edge и др.).
2. Выберите нужный модуль: **Очистка**, **Дедупликация** или **Экстрактор**.
3. Вставьте исходный текст или список в поле ввода.
4. Отметьте нужные параметры (для очистки) или выберите режим сортировки (для дедупликации).
5. Нажмите кнопку обработки, затем скопируйте результат или скачайте его в файл.

> 💡 *Инструмент готов к работе сразу после открытия файла. Интернет-соединение не требуется для обработки данных.*
